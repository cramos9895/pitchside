## ⚽ Competition Constants

### Match Styles
- `Pickup`: Casual matches with individual signups.
- `Tournament`: Bracket-based competitive play.
- `League`: Multi-week structured competition.
- `Rolling League`: Flexible league format without strict scheduling.

### Surface Types
- `Turf (Indoor)`
- `Turf (Outdoor)`
- `Grass`
- `Futsal / Hardwood`

### Game Formats
- `5v5`, `6v6`, `7v7`, `8v8`, `11v11`
